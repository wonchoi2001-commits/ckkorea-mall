# CKKorea Mall

씨케이코리아용 Next.js 자사몰 프로젝트입니다.

## 실행 방법

1. Node.js 20+ 설치
2. 의존성 설치
   ```bash
   npm install
   ```
3. 환경변수 설정
   ```bash
   cp .env.local.example .env.local
   ```
4. 개발 서버 실행
   ```bash
   npm run dev
   ```
5. 브라우저에서 `http://localhost:3000` 접속

## 배포
- GitHub에 업로드
- Vercel 프로젝트 생성
- `www.ckkorea.co.kr` 도메인 연결
- 네이버 서치어드바이저 등록
